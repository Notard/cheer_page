# 딴짓 포트폴리오

작게 만든 웹 딴짓을 카드 격자로 모아 보여주는 정적 포트폴리오입니다.

## 딴짓 추가하기

1. 새 웹 프로젝트 폴더를 `projects/새-이름/` 아래에 넣습니다.
2. `projects.js`에 항목을 하나 추가합니다.

```js
{
  title: "새 딴짓 제목",
  description: "한 줄 설명을 여기에 씁니다.",
  href: "./projects/새-이름/",
}
```

삭제할 때는 `projects.js`에서 해당 항목을 지우고, 필요하면 `projects/` 안의 폴더도 지우면 됩니다.
