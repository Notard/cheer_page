# 응원 페이지 배포 안내

이 폴더 안의 `index.html`, `styles.css`, `script.js`, `warm-background.png` 네 파일을 그대로 올리면 됩니다.

## 추천: GitHub Pages

1. GitHub에서 새 저장소를 만듭니다. 예: `cheer-page`
2. 이 폴더의 파일 4개를 저장소 최상위에 업로드합니다.
3. 저장소의 `Settings` -> `Pages`로 갑니다.
4. `Build and deployment`에서 `Deploy from a branch`를 고릅니다.
5. Branch를 `main`, 폴더를 `/root`로 선택하고 저장합니다.
6. 몇 분 뒤 `https://사용자이름.github.io/cheer-page/` 주소가 생깁니다.

GitHub 계정과 저장소를 유지하는 한 이 주소는 계속 유지됩니다.

## 이름과 응원 문구 바꾸기

페이지 오른쪽 아래 입력칸에서 이름과 문구를 바꾼 뒤 `공유 주소 복사`를 누르면 됩니다.

직접 주소를 만들 수도 있습니다.

```text
https://사용자이름.github.io/cheer-page/?name=수연&message=오늘도%20정말%20잘하고%20있어
```

## 더 쉬운 임시 배포: Netlify Drop

로그인한 상태에서 <https://app.netlify.com/drop>에 이 폴더를 끌어다 놓으면 바로 주소가 생깁니다. 다만 가장 오래 안정적으로 쓰려면 GitHub Pages나 Netlify 계정에 연결된 사이트로 유지하는 편이 좋습니다.
